POS Custom Order Sequence Odoo App

Point of Sales Custom Receipt Sequence Odoo Apps

POS Order Personalized Sequence Odoo Apps

POS Order Custom Receipt Sequence Odoo App helps users to manage and organize receipts with customized sequence number for point of sale operation. Users can easily customize their receipts number and set them up in a specific sequence that suits business needs. Users have the option to enable or disable custom sequence for receipt and print, also visible in order view.

Features

Enable/Disable Custom Sequence Option

User have option to enable or disable custom sequence option for pos receipt.

Print Custom Sequence in Receipt and Order

User can see custom pos order receipt sequence in receipt report and order view.
Demo Request Video Tutorial

Custom Receipt Sequence Configuration

Under Point of Sale > Configuration > Settings menu user need to enable "Set receipt sequence number" option and select sequence for point of sale order receipt.

Custom Receipt Sequence

User can see created custom sequence for pos order receipt number.

POS Custom Receipt Sequence

User can see generated point of sale order receipt with custom sequence number.

POS Custom Receipt Sequence on Order

User can also see generated pos order custom sequence number on order view.
